## Важно: запуск `install.ps1` может быть заблокирован политикой выполнения (ExecutionPolicy)

На некоторых системах PowerShell запрещает выполнение локальных `.ps1` файлов по умолчанию.  
В этом случае при запуске `install.ps1` появляется ошибка:

```text
Невозможно загрузить файл ...install.ps1, так как выполнение сценариев отключено в этой системе.
```
